# MEDIXIA MARKET

Une vitrine web statique pour le projet MEDIXIA MARKET, un site proposant des équipements médicaux, accessoires et consommables pour les professionnels de santé.

## Pages du site

- `index.html` : page d'accueil avec la proposition de valeur et les atouts du projet.
- `produits.html` : catalogue des produits disponibles.
- `apropos.html` : présentation du projet et de la mission.
- `contact.html` : page de contact pour prendre rendez-vous ou demander des informations.

## Fichiers importants

- `style.css` : styles CSS du site.
- `script.js` : scripts front-end (navigation, interactions, etc.).
- `produits.json` : données des produits pouvant être utilisées pour alimenter la page `produits.html`.

## Comment utiliser

1. Ouvrir le fichier `index.html` dans un navigateur web.
2. Naviguer entre les pages via le menu principal.
3. Modifier le contenu HTML, CSS ou JSON selon vos besoins.

## Technologies

- HTML
- CSS
- JavaScript

## Objectif

Ce projet présente une maquette de site pour un commerce de matériel médical, avec une interface simple et responsive pour découvrir l'offre, en savoir plus sur l'entreprise, et contacter l'équipe.
